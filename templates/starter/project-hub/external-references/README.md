# External References

This folder contains cached copies of authoritative external references.

## Purpose

- Store important external documents (RFCs, specs, standards)
- Preserve references that may change or disappear
- Provide offline access to critical documentation

## Guidelines

- Include source URL and retrieval date
- Use for references that inform project decisions
- Don't use for general reading material

## Naming Convention

```
[YYYY-MM-DD]-[source]-[title].[ext]
```

Example: `2026-01-15-ietf-rfc7231-http-semantics.pdf`
