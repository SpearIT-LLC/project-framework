# Proof of Concept

This folder contains proof-of-concept code and experiments.

## Guidelines

- POC code is exploratory and not production-ready
- Document findings and conclusions
- Move successful POCs to `src/` when ready for production
- Archive or delete failed POCs

## Structure

```
poc/
├── README.md           # This file
├── experiment-name/    # Individual POC folders
│   ├── README.md      # POC description and findings
│   └── ...            # POC code and assets
```
