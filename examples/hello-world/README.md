# Project: Hello World

A simple demonstration project using the Standard Project Framework.

## Overview

This is a minimal "Hello World" application that demonstrates the structure and organization of a project built with the Standard Project Framework. It serves as a reference implementation and starting point for new projects.

## Quick Start

```bash
# Run the application
node src/hello-world.js

# Run tests (requires Jest)
npm test
```

## Project Structure

```
project-hello-world/
├── src/              # Application source code
├── tests/            # Test files
├── docs/             # Project documentation
└── thoughts/         # Project tracking and decision history
```
